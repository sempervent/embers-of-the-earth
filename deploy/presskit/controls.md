# Embers of the Earth - Controls

## Default Controls

### Movement

- **WASD** / **Arrow Keys** - Move player
- **Space** - Interact / Confirm
- **E** - Action (till soil, plant crop, harvest, talk to NPC)

### UI Navigation

- **ESC** - Open menu / Cancel
- **M** - Open map
- **Tab** - Toggle inventory
- **I** - Open inventory
- **J** - Open journal
- **C** - Open character sheet
- **F** - Open family tree

### Time Management

- **T** - Advance day
- **Pause** - Pause game

### Camera

- **Mouse** - Pan camera (if enabled)
- **Scroll Wheel** - Zoom in/out (if enabled)

### Debug (Debug builds only)

- **F3** - Toggle debug overlay
- **F4** - Toggle performance profiler
- **F5** - Quick save
- **F9** - Quick load
- **`** (Backtick) - Open dev console

## Controller Support (if implemented)

### Movement

- **Left Stick** - Move player
- **A** / **X** - Interact / Confirm
- **B** / **Circle** - Cancel / Menu

### UI Navigation

- **D-Pad** - Navigate menus
- **Start** - Open menu
- **Select** - Toggle inventory
